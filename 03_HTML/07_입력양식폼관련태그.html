사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할을 수행

form action = "" method = ""

*action : 해당 form에 사용자가 입력한 값들을 전달하면서 요청을 보낼 서버의 경로 *method : 요청 전송방식을 지정하는 속성(get(기본값), post) > get : 요청시 사용자가 입력한 값들을 head에 담아 전송한다. -> url에 값이 전부 노출 > post : 요청시 사용자가 입력한 값들을 url에 노출되지 않는 형식으로 전달. form내에 sumbit버튼 클릭 시 action에 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨

selectBox처럼 여러개 중 하나를 선택할 때 주로 사용

checkbox

여러개를 선택할 때 사용

다중 선택 시 주로 사용함
name 속성을 통해서 여러개의 체크박스를 하나의 선택지로 묶을 수 있음
checked속성을 통해서 체크박스를 체크된 상태로 변경할 수 있음
id - for 속성을 통해서 input과 label태그를 묶을 수 있음
서버에 실제로 전달될 때 key - name 속성값, value -> value속성값으로 전달

| button : 클릭할 수 있는 버튼을 정의한다.
| checkbox : 체크박스를 정의한다.
| color : 색을 선택할 수 있는 입력 필드를 정의한다.
| date : 날짜를 선택할 수 있는 입력필드를 정의한다.
| datetime : 날짜와 시간을 입력할 수 있는 필드(현재 대부분의 브라우저에서 지원되지 않는다. UTC기준)
| datetime-loacl 날짜와 시간을 선택할 수 있는 입력 필드를 정의한다.(현재 대부분의 브라우저에서 지원된다. 사용자 로컬 시간 기준)
| email : 이메일 주소를 입력할 수 있는 입력 필드를 정의한다.
| file : 업로드할 파일을 선택할 수 있는 입력 필드와 "파일 선택"버튼을 정의한다.
| hidden : 사용자에게는 보이지 않는 숨겨진 입력필드를 정의한다.
| image : 제출 버튼(submit button)으로 사용될 이미지를 정의한다.
| month : 날짜를 선택할 수 있는 입력 필드를 정의한다.(year, month)
| number : 숫자를 입력할 수 있는 입력 필드를 정의한다.
| password : 비밀번호를 입력할 수 있는 입력 필드를 정의한다.
| radio : 라디오버튼(radio)을 정의한다.(중복체크를 허용하지 않음)
| range : 슬라이드 바를 조정하여 범위 내의 숫자를 선택할 수 있는 입력 필드를 정의한다.
| value : input 요소의 값을 명시한다.
| submit : 제출 버튼을 정의한다.
| search : 검색어를 입력할 수 있는 텍스트 필드를 정의한다.
| type 속성의 기본값으로, 한 줄로 된 텍스트 필드를 정의한다.
| tel : 전화번호를 입력할 수 있는 입력 필드를 정의한다.
| time : 시간을 선택할 수 있는 입력 필드를 정의한다.
| url : URL 주소를 입력할 수 있는 입력 필드를 정의한다.
| week : 날짜를 선택할 수 있는 입력 필드를 정의한다.(year, week)
